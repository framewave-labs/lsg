## 1. fal.ai 영상 생성 비동기 흐름

### 1.1 전체 흐름도

```
[사용자] 프롬프트 입력 + 생성 버튼 클릭
        │
        ▼
[FE → BE] POST /api/generate-video
        │
        ▼
[BE → fal.ai] queue.submit()
        │ 응답: { request_id: "xxx" }
        │
        ▼
[BE → FE] { requestId: "xxx", status: "IN_QUEUE" } 반환
        │
        ▼
[FE] 5초 간격으로 폴링 시작
[FE → BE] GET /api/video-status?requestId=xxx
        │
        ├── status: "IN_QUEUE"   → 계속 폴링 (큐 대기 중)
        ├── status: "IN_PROGRESS" → 계속 폴링 (생성 중)
        └── status: "COMPLETED"  → 폴링 종료
              │
              ▼
        [BE → fal.ai] queue.result() 호출
              │
              ▼
        [BE → FE] { videoUrl: "https://..." } 반환
              │
              ▼
        [FE] 비디오 플레이어에 URL 세팅 + 자동 재생
```

### 1.2 사용자 화면 기준 상태 정의

| fal.ai 상태 | 사용자가 보는 것 | UI 표시 |
|------------|--------------|--------|
| IN_QUEUE | "대기 중입니다. 잠시만 기다려주세요." | 스피너 + 텍스트 |
| IN_PROGRESS | "영상을 생성하고 있습니다..." | 프로그레스 바 (가짜 진행, 0→80%) |
| COMPLETED | 영상 플레이어 출력 | 비디오 플레이어 자동 표시 |
| ERROR | "생성에 실패했습니다. 다시 시도해주세요." | 에러 메시지 + 재시도 버튼 |

> **PM 요청:** 영상 생성은 통상 30초~2분 소요. 사용자가 기다리는 동안 진행 중임을 체감할 수 있도록 IN_PROGRESS 상태에서는 가짜 프로그레스 바(0%→80%까지만)를 표시하고, 완료 시 100%로 점프 후 플레이어 전환. 타임아웃은 10분 기준.

### 1.3 타임아웃 처리

- **10분 경과 후에도 COMPLETED 미수신 → ERROR 처리**
- 에러 메시지: "영상 생성 시간이 초과되었습니다. 다시 시도해 주세요."
- 재시도 버튼 → 새 request_id로 처음부터 재시작

### 1.4 비용 통제 규칙 (PM 요청 사항)

| 환경 | 모델 | 영상 길이 | 이유 |
|------|------|---------|------|
| 개발·테스트 | **Kling 3.0** | **5초** | $0.55/클립 — 비용 최소화 |
| 데모·최종 발표 | **Seedance 2.0** | **10초** | $3.00/클립 — 품질 최우선 |

> fal.ai 대시보드에서 Hard Limit **$120** 설정 필수. 킥오프(6/30) 당일 BE가 확인.

### 1.5 fal.ai 코드 참고

```typescript
import { fal } from '@fal-ai/client';
fal.config({ credentials: process.env.FAL_KEY });

// 1) 제출
const { request_id } = await fal.queue.submit(
  'fal-ai/kling-video/v3/pro/text-to-video',
  { input: { prompt, duration: 5 } }
);

// 2) 상태 조회 (FE가 5초 간격으로 GET /api/video-status 호출)
const status = await fal.queue.status(
  'fal-ai/kling-video/v3/pro/text-to-video',
  { requestId: request_id }
);
// status.status: "IN_QUEUE" | "IN_PROGRESS" | "COMPLETED" | "FAILED"

// 3) 완료 시 결과 조회
const result = await fal.queue.result(
  'fal-ai/kling-video/v3/pro/text-to-video',
  { requestId: request_id }
);
// result.data.video.url → 영상 URL
```

## 2. Claude Sonnet 역프롬프트 — 시스템 프롬프트 설계

### 2.1 기능 개요 (F004)

- **입력:** 레퍼런스 이미지 (사용자 업로드)
- **처리:** Claude Sonnet 멀티모달로 이미지 분석 → 재현용 프롬프트 추출
- **출력:** JSON (분석 항목 + 통합 영문 프롬프트)

### 2.2 시스템 프롬프트 (v1)

```
You are an expert in visual analysis and AI image prompt engineering for broadcast production.

Analyze the provided image and extract a reproduction prompt that can regenerate a similar image using AI image generation tools (like GPT Image 2 or Midjourney).

Output format (strict JSON only, no extra text):
{
  "analysis": {
    "style": "overall visual style (e.g., cinematic, film noir, documentary, commercial, anime)",
    "lighting": "lighting description (e.g., golden hour, dramatic side lighting, soft diffused, hard light, neon)",
    "composition": "framing and composition (e.g., close-up, wide shot, rule of thirds, symmetrical, dutch angle)",
    "lens": "lens/camera feel (e.g., shallow depth of field, wide angle distortion, telephoto compression, 85mm portrait)",
    "color_palette": "color description (e.g., warm orange tones, desaturated blue, high contrast black and white, pastel)",
    "mood": "emotional atmosphere (e.g., melancholic, tense, romantic, energetic, mysterious)"
  },
  "prompt_en": "complete reproduction prompt in English (80 words or less), incorporating all analysis elements above, ending with: photorealistic, high detail"
}
```

### 2.3 출력 예시

```json
{
  "analysis": {
    "style": "cinematic, Korean drama",
    "lighting": "soft natural window light, overcast sky diffusion",
    "composition": "medium shot, slightly high angle, subject off-center (rule of thirds)",
    "lens": "shallow depth of field, 85mm portrait lens feel, background bokeh",
    "color_palette": "cool blue-grey tones, desaturated, slight teal shadow",
    "mood": "melancholic, introspective, quiet"
  },
  "prompt_en": "a young woman sitting by a large window in a modern apartment, medium shot from slightly above, soft diffused overcast light from window, cool blue-grey color palette, shallow depth of field with bokeh background, melancholic and introspective mood, Korean drama cinematic style, 85mm portrait lens, photorealistic, high detail"
}
```

### 2.4 API Route 구현 포인트

```typescript
// /api/reverse-prompt/route.ts
export async function POST(req: Request) {
  const { imageBase64, mimeType } = await req.json();
  
  const response = await client.messages.create({
    model: 'claude-sonnet-4-5',
    max_tokens: 1024,
    system: REVERSE_PROMPT_SYSTEM_PROMPT,
    messages: [{
      role: 'user',
      content: [
        {
          type: 'image',
          source: { type: 'base64', media_type: mimeType, data: imageBase64 }
        },
        { type: 'text', text: '이 이미지를 분석해 재현용 프롬프트를 추출해주세요.' }
      ]
    }]
  });
  
  const result = JSON.parse(response.content[0].text);
  return Response.json(result);
}
```

> **PM 참고 (UX):** 역프롬프트 결과 화면에서 `analysis` 항목(스타일·조명·구도·렌즈·색감·분위기)을 태그 형태로 시각화해 주면 사용자가 어떤 요소가 추출됐는지 바로 파악 가능. `prompt_en`은 복사 버튼 + [이미지 생성으로 이어서] 버튼과 함께 표시.

## 3. API Route 목록 및 입출력 명세

| Method | Route | 기능 | 담당 주차 |
|--------|-------|------|---------|
| POST | `/api/generate-image` | 이미지 생성 (GPT Image 2 / Gemini) | 2주차 |
| POST | `/api/generate-video` | 영상 생성 (fal.ai queue submit) | 3주차 |
| GET | `/api/video-status` | 영상 생성 상태 조회 (polling) | 3주차 |
| POST | `/api/reverse-prompt` | 역프롬프트 추출 (Claude 멀티모달) | 3주차 도전 |
| POST | `/api/library` | 미디어 저장 (Supabase) | 2주차 |
| GET | `/api/library` | 미디어 조회 (필터·페이지네이션) | 2주차 |
