## 이미지 QA 체크리스트

### 1. 자연어 프롬프트 변환

- [ ] 한국어 입력의 원래 의도가 영어 변환 후에도 유지되는가
- [ ] PM이 직접 최적화한 프롬프트 vs 실제 서비스 Claude 변환 결과 비교 — 갭이 크면 시스템 프롬프트 튜닝 필요
- [ ] "프롬프트 향상" 토글 On/Off 결과 차이가 실제로 유의미한가
- [ ] 변환 실패 시 원문으로 폴백되는가, 아니면 생성 자체가 막히는가
- [ ] 변환 처리 시간이 사용자가 체감할 정도로 느리지 않은가

### 2. 이미지 결과물 품질

- [ ] 프롬프트 반영도 — 핵심 피사체·분위기가 실제로 표현됐는가
- [ ] 해상도 — 1024px 이상, 흐릿함 없음
- [ ] 구도 — 주요 피사체가 잘리지 않았는가
- [ ] 캐릭터 일관성 — 메뉴 B 키이미지, 동일 인물 여러 장 생성 시 얼굴형·헤어·의상 유지되는가
- [ ] **의도치 않은 텍스트/숫자/로고 렌더링 여부** — "no text" 지시가 있어도 프롬프트 내 반복 토큰(예: 번호 라벨)이 있으면 새어나올 수 있음 (실제 발생 이력: 3x3 그리드 테스트에서 "Panel 1~9" 표기가 이미지에 그대로 렌더링됨)
- [ ] 다중 요소(레퍼런스 여러 장, 그리드 등) 조합 시 소품·배경 연속성 붕괴 없는가 (실제 발생 이력: 그리드 특정 패널에만 미리 선언하지 않은 의자가 등장)
- [ ] 생성 성공률 — 5회 중 3회 이상 Pass 기준 충족

### 3. 옵션값 반영

- [ ] 비율(1:1/16:9/9:16/4:3) 선택값과 결과 메타데이터 일치 — 4:3은 GPT Image 2 지원 여부 별도 확인 필요
- [ ] 품질(medium/high) 반영 확인
- [ ] 레퍼런스 첨부(Style/Character/Add, 최대 8장, PNG·WEBP·JPG, 50MB 이하) 검증 및 반영 정확도

### 4. 라이브러리 연동

- [ ] 생성 즉시 자동저장 (별도 저장 버튼 없음)
- [ ] 재생성 시 기존 결과 유지 + 신규 항목 별도 저장
- [ ] 찜하기 토글 — 즉시 반영, 실패 시 이전 상태로 롤백
- [ ] 필터·정렬 (전체/이미지/영상/찜한 것, 최신순) 동작
- [ ] 상세 모달 — 프롬프트·모델·비율·생성일시 정확히 표시
- [ ] 다운로드 — 파일명 규칙 준수, 원본 해상도
- [ ] "이 이미지를 영상 생성에 사용" 연동 동작

## 영상 QA 체크리스트

### 1. 3x3 그리드 이미지 입력

- [ ] 9칸 전부 누락 없이 생성되는가
- [ ] 좌상단→우하단이 하나의 스토리 순서로 인식되는가 (번호 라벨 없이도)
- [ ] 번호·텍스트·로고 미노출 확인
- [ ] 9칸 전체에서 캐릭터·배경·조명 일관성 유지되는가
- [ ] 공통 배경 지시와 개별 패널 지시가 서로 모순되지 않는가 (생성 전 프롬프트 자체 검토)

### 2. Kling 3.0 (1차 테스트용)

- [ ] 그리드를 레퍼런스로 넣었을 때 **start 프레임(1번 위치)에서 시작 / end 프레임(9번 위치)에서 종료**가 의도대로 되는가 — 현재 알려진 취약 지점
- [ ] 그리드가 그대로 노출되지 않고 실제로 슬라이스되는가
- [ ] 최대 6샷 제한 — 9샷을 2회로 나눠 테스트했을 때 이어붙인 결과가 자연스러운가
- [ ] 대사가 필요한 씬에서 립싱크·발화 타이밍 정확도

### 3. Seedance 2.0 (최종 품질 기준)

- [ ] Duration 15초 준수, 9샷 타임코드 합이 정확히 맞는가
- [ ] Shot 라벨 인식 — 여러 샷이 뭉개지지 않고 각각 구분되는가
- [ ] 카메라 무브먼트 반영 — 지정값대로 구현되는가(정적 지정 시 정적 유지되는가)
- [ ] 사운드 반영 — 지정한 음향이 제네릭 배경음으로 대체되지 않는가
- [ ] 오디오 포함 필수 요건(C팀 조건) 충족 — generate_audio 옵션값, 비용 영향 확인
- [ ] 일관성 — 인물·장소·색감이 샷 간 급변하지 않는가
- [ ] "no readable text/logos/numbers" 지정 시 실제로 텍스트 안 나오는가

### 4. 상태·에러 처리

- [ ] 폴링 규칙(5초 간격, 최대 120회/10분) 동작
- [ ] 일시적 폴링 실패(1~2회)로 에러 전환되지 않는가 (3회 연속만 전환)
- [ ] 10분 타임아웃 처리
- [ ] 비용 한도 초과 처리
- [ ] 재생성 시 기존 결과 유지 + 신규 항목 별도 저장

### 5. 결과 표시·라이브러리 연동

- [ ] completed 시점에 자동저장되는가
- [ ] 재생 정상 동작(controls, 자동재생 여부)
- [ ] 메타정보(모델·길이·생성일시) 정확히 표시
- [ ] 액션버튼(찜하기/재생성/다운로드) 동작
- [ ] 통합 라이브러리에서 이미지·영상 혼합 조회 정상
