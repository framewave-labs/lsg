# A2팀 IA v0.6
> 작성일: 2026-07-12 | 작성자: 이슬기 (PM)
> 기준: 로그인 후 핵심 기능 바로 노출 / 단일 페이지 + 기능별 화면 구조
> **v0.6 진행 중 — 손찬용 PM과 월요일 통합 예정 (최종본 아님, 각자 작성 후 병합)**
> v0.5 대비 변경: **라이브러리 저장 방식 정정** — 생성 결과는 [저장] 버튼 없이 즉시 자동으로 라이브러리(=히스토리)에 저장됨. "찜하기"는 저장 액션이 아니라 라이브러리 내에서 마음에 드는 항목만 따로 모아보는 **즐겨찾기 토글**로 역할 재정의
> 개발자 컨펌 전 — 검토 후 확정 예정

## 전제 조건

| 항목 | 결정 내용 |
|------|---------|
| 온보딩/히어로 섹션 | 없음 — 로그인 후 핵심 기능 바로 노출 |
| 화면 구조 | 단일 페이지 + 기능별 화면 (모달·팝업) |
| 로그인 방식 | 기본 로그인만 (소셜 로그인 없음) |
| 유료플랜·이미지 편집기 | 이번 범위 제외 → 고도화 로드맵 |
| 이미지 생성 플로우 | **2단계 분리** — 메뉴 A(초기 레퍼런스) → 메뉴 B(키이미지) |
| AI 프롬프트 변환 토글 | 메뉴 A·B에만 존재. 미리보기 없이 On 시 백그라운드 변환 후 바로 생성 요청 |
| 라이브러리 저장 방식 | **생성 결과는 예외 없이 즉시 자동 저장** — 라이브러리 = 히스토리. 별도 [저장] 버튼 없음 |
| 찜하기 | 저장 액션 아님 — 라이브러리에 이미 있는 항목 중 마음에 드는 것만 표시하는 **즐겨찾기 토글** (하트 아이콘) |

## 화면 구조 (IA)

```
[비로그인]
├── 로그인
└── 회원가입

[로그인 후]
└── 홈 화면
    ├── (좌측 사이드바) ← 미확정
    │   ├── 홈
    │   ├── 이미지
    │   ├── 영상
    │   └── 라이브러리
    │
    ├── 홈 대시보드
    │   ├── 서비스 간단 소개
    │   ├── 이미지 만들기 진입
    │   ├── 영상 만들기 진입
    │   ├── 최근 작업물
    │   └── 라이브러리 바로가기
    │
    ├── 이미지 생성 화면
    │   │
    │   ├── [메뉴 A] 초기 레퍼런스 생성          (M001~M006)
    │   │   ├── 프롬프트 입력창 (한국어/영어 자유 입력)
    │   │   ├── AI 프롬프트 변환 토글 (On/Off)
    │   │   │   ├── Off: 입력한 텍스트 그대로 생성 요청
    │   │   │   └── On: 백그라운드에서 영어로 변환 후 생성 요청 (미리보기 없음)
    │   │   ├── [생성] 버튼 / Enter
    │   │   ├── 옵션 선택 (비율·품질)
    │   │   ├── 생성 중 상태
    │   │   └── 생성 결과
    │   │       ├── 배경 이미지 1장
    │   │       └── 캐릭터 다각도 (정면·측면·후면)
    │   │
    │   └── [메뉴 B] 키이미지 생성               (A001)
    │       ├── 상세 프롬프트 입력 — 자유 텍스트 + 속성 선택 사이드바 둘 다 지원 (확정)
    │       │   ├── 자유 텍스트 입력 (PD 표준형 PART1+PART2 구조)
    │       │   └── 속성 선택 사이드바
    │       │       ├── 파트1 캐릭터 기본
    │       │       ├── 파트2 아트 스타일·세계관
    │       │       ├── 파트3 헤어
    │       │       ├── 파트4 얼굴
    │       │       └── 파트5 의상·악세서리 (컬러 항목은 프리셋 아닌 컬러 팔레트+헥스코드 직접 입력)
    │       ├── AI 프롬프트 변환 토글 (On/Off) — 자유 텍스트 입력 시에만 적용
    │       │   ├── Off: 입력한 텍스트 그대로 생성 요청
    │       │   └── On: 백그라운드에서 영어로 변환 후 생성 요청 (미리보기 없음)
    │       ├── [생성] 버튼 / Enter
    │       ├── 생성 중 상태
    │       └── 생성 결과
    │           └── 완성도 높은 캐릭터 시트
    │               (표정·소품·컬러 팔레트·다각도 포함)
    │
    │   ── 생성 완료 시 결과는 별도 조작 없이 즉시 라이브러리에 자동 저장됨
    │   ── 공통 액션 (메뉴 A·B 모두)
    │       ├── 찜하기 (즐겨찾기 토글 — 저장 액션 아님, 이미 자동 저장된 항목을 표시만 함)
    │       ├── 다운로드
    │       ├── 프롬프트 복사
    │       ├── 재편집
    │       ├── 삭제
    │       └── 재생성
    │
    ├── 영상 생성 화면                        (A002~A005)
    │   ├── B서비스에서 받은 9개 컷 (3×3 그리드 1장) 이미지 업로드
    │   ├── 영상 프롬프트 입력
    │   ├── 옵션 선택
    │   │   ├── 모델 (Seedance 2.0 기본값 / Kling 3.0 — 대사·저비용 테스트용 대안)
    │   │   └── 길이 (모델별 상이 — Seedance 2.0: 4~15초 / Kling 3.0: 3~15초)
    │   ├── 생성 요청
    │   ├── 생성 중 상태
    │   │   ├── queued (대기 중)
    │   │   ├── processing (진행 중)
    │   │   ├── completed (생성 완료)
    │   │   └── failed (생성 실패)
    │   ├── 영상 결과 미리보기 — 생성 완료 시 즉시 라이브러리에 자동 저장됨
    │   ├── 찜하기 (즐겨찾기 토글 — 저장 액션 아님)
    │   ├── 다운로드
    │   ├── 프롬프트 복사
    │   ├── 재편집
    │   ├── 삭제
    │   └── 재생성
    │
    ├── 라이브러리 화면                       (A006) — 모든 생성 결과가 자동 저장되는 히스토리
    │   ├── 이미지·영상 혼합 그리드
    │   ├── 필터 (전체·이미지·영상·찜한 것만 (제안))
    │   ├── 정렬 (최신순)
    │   └── 작업물 상세
    │       ├── 이미지 또는 영상 크게 보기
    │       ├── 생성 당시 프롬프트 확인
    │       ├── 찜하기 (즐겨찾기 토글)
    │       ├── 다운로드
    │       ├── 다시 생성하기
    │       ├── 영상 생성에 사용하기
    │       └── 삭제
    │
    ├── 역프롬프트 화면                       (S001~S004) ← S등급 도전
    │   ├── 이미지 업로드
    │   ├── 이미지 분석 결과
    │   ├── 프롬프트 추출
    │   ├── 프롬프트 수정
    │   └── 재생성 연결 → 이미지 생성 화면으로
    │
    ├── 내 프로필·설정
    │   ├── 계정 정보
    │   ├── 구독 상태
    │   ├── 문의
    │   ├── 기본 생성 옵션
    │   └── 로그아웃
    │
    ├── 사전 프롬프트 가이드 (룰 설정)       ← 진입점 미확정
    │   └── 네거티브 키워드 등
    │
    └── 공유                                  ← 진입점 미확정
        └── 링크 공유 (다운로드 없음 / 영상·사진 단독 뷰 + 서비스 로고·링크)
```
