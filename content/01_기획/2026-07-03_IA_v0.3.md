# A2팀 IA v0.3
> 작성일: 2026-07-03 | 작성자: 이슬기 (PM) · UX/UI(이예린) 1차 협의 반영
> 기준: 로그인 후 핵심 기능 바로 노출 / 단일 페이지 + 기능별 화면 구조
> **개발자 컨펌 전 — 검토 후 확정 예정**

## 전제 조건

| 항목 | 결정 내용 |
|------|---------|
| 온보딩/히어로 섹션 | 없음 — 로그인 후 핵심 기능 바로 노출 |
| 화면 구조 | 단일 페이지 + 기능별 화면 (모달·팝업) |
| 로그인 방식 | 기본 로그인만 (소셜 로그인 없음) |

| 고도화 통합 | 올인원 통합 시 히어로 섹션·온보딩 추가 (추후) |

## 화면 구조 (IA)

```
[비로그인]
├── 로그인
└── 회원가입

[로그인 후]
└── 홈 화면
    ├── (좌측 사이드바) ← 미확정
    │   ├── 홈
    │   ├── 이미지
    │   ├── 영상
    │   └── 라이브러리
    │
    ├── 홈 대시보드
    │   ├── 서비스 간단 소개
    │   ├── 이미지 만들기 진입
    │   ├── 영상 만들기 진입
    │   ├── 최근 작업물
    │   └── 라이브러리 바로가기
    │
    ├── 이미지 생성 화면                      (M001~M006)
    │   ├── 생성 목적 선택
    │   │   ├── 캐릭터
    │   │   ├── 배경
    │   │   └── 키이미지 (배경 시트·캐릭터 시트)
    │   ├── 프롬프트 입력
    │   ├── 옵션 선택
    │   │   ├── 비율 (크기·포맷)
    │   │   └── 품질
    │   ├── 생성 중 상태
    │   ├── 생성 결과
    │   ├── 방향별 레퍼런스                   (A001) ← 이미지 생성 화면 안에 탭/섹션으로 포함
    │   │   ├── 기준 이미지 선택
    │   │   ├── 캐릭터 시트
    │   │   ├── 배경 시트
    │   │   └── 필요 시 키이미지 생성
    │   ├── 찜하기 (라이브러리 저장)
    │   ├── 다운로드
    │   ├── 프롬프트 복사
    │   ├── 재편집
    │   ├── 삭제
    │   └── 재생성
    │
    ├── 영상 생성 화면                        (A002~A005)
    │   ├── B서비스에서 받은 9개 컷 (3×3 그리드 1장) 이미지 업로드
    │   ├── 영상 프롬프트 입력
    │   ├── 옵션 선택
    │   │   ├── 모델 (Seedance 2.0 / Kling 3.0)
    │   │   └── 길이 (3~15초)
    │   ├── 생성 요청
    │   ├── 생성 중 상태
    │   │   ├── queued (대기 중)
    │   │   ├── processing (진행 중)
    │   │   ├── completed (생성 완료)
    │   │   └── failed (생성 실패)
    │   ├── 영상 결과 미리보기
    │   ├── 찜하기 (라이브러리 저장)
    │   ├── 다운로드
    │   ├── 프롬프트 복사
    │   ├── 재편집
    │   ├── 삭제
    │   └── 재생성
    │
    ├── 라이브러리 화면                       (A006)
    │   ├── 이미지·영상 혼합 그리드
    │   ├── 필터
    │   │   ├── 전체
    │   │   ├── 이미지
    │   │   └── 영상
    │   ├── 정렬 (최신순)
    │   └── 작업물 상세
    │       ├── 이미지 또는 영상 크게 보기
    │       ├── 생성 당시 프롬프트 확인
    │       ├── 다운로드
    │       ├── 다시 생성하기
    │       ├── 영상 생성에 사용하기
    │       └── 삭제
    │
    ├── 역프롬프트 화면                       (S001~S004) ← S등급 도전
    │   ├── 이미지 업로드
    │   ├── 이미지 분석 결과
    │   ├── 프롬프트 추출
    │   ├── 프롬프트 수정
    │   └── 재생성 연결 → 이미지 생성 화면으로
    │
    ├── 내 프로필·설정
    │   ├── 계정 정보
    │   ├── 구독 상태
    │   ├── 문의
    │   ├── 기본 생성 옵션
    │   └── 로그아웃
    │
    ├── 사전 프롬프트 가이드 (룰 설정)       ← 진입점 미확정
    │   └── 네거티브 키워드 등
    │
    └── 공유                                  ← 진입점 미확정
        └── 링크 공유 (다운로드 없음 / 영상·사진 단독 뷰 + 서비스 로고·링크)
```

## 미확정 항목

| 항목 | 현황 | 확정 시점 |
|------|------|---------|
| 사이드바 구조·디자인 | 좌측 사이드바 방향으로 논의 중 | UX/UI 와이어프레임 확정 후 |
| 역프롬프트 진입점 위치 | 사이드바 포함 여부 미정 | 와이어프레임 확정 후 |
| 사전 프롬프트 가이드 진입점 | 프로필 내 or 생성 화면 내 미정 | 와이어프레임 확정 후 |
| 공유 기능 진입점 | 생성 결과 또는 라이브러리 내 미정 | 와이어프레임 확정 후 |
| 라이브러리 방향 | 현재 히스토리 / 차후 좋아요·스크랩 (BE 안건) | 개발자 컨펌 후 |
