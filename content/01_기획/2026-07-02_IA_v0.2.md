## 화면 구조 (IA)

```
[비로그인]
├── 로그인
└── 회원가입

[로그인 후]
└── 홈 화면
    ├── (좌측 사이드바) ← 미확정
    │   ├── 이미지
    │   ├── 영상
    │   └── 라이브러리
    │
    ├── 이미지 생성 화면       (M001~M006)
    │   ├── 프롬프트 입력
    │   ├── 옵션 선택 (비율·품질)
    │   ├── 방향별 레퍼런스    (A001) ← 이미지 생성 화면 안에 포함
    │   │   ├── 캐릭터 시트
    │   │   └── 배경 시트
    │   ├── 생성 결과
    │   └── 저장·재생성
    │
    ├── 영상 생성 화면         (A002~A005)
    │   ├── 프롬프트 입력
    │   ├── 옵션 선택 (모델·길이)
    │   ├── 생성 중 (진행 상태)
    │   ├── 생성 결과
    │   └── 저장·재생성
    │
    ├── 라이브러리 화면        (A006)
    │   ├── 이미지·영상 혼합 그리드
    │   ├── 필터 (전체·이미지·영상)
    │   └── 상세 팝업 (다운로드·삭제)
    │
    ├── 역프롬프트 화면        (S001~S004) ← S등급 도전
    │   ├── 이미지 업로드
    │   ├── 분석 결과 (태그)
    │   ├── 프롬프트 추출
    │   └── 재생성 연결 → 이미지 생성 화면으로
    │
    └── 내 프로필·설정
```
