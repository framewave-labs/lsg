**레퍼런스 서비스 분석 — A2팀 관점**

**1. Krea.ai**

한 줄 요약: 실시간 이미지 생성 캔버스 + 64개 모델 통합 플랫폼

핵심 UX 특징
- **Realtime Canvas:** 왼쪽에 스케치/업로드 → 오른쪽에 AI가 50ms 이내 즉시 렌더링
- **모델 애그리게이터:** Flux, Runway, Luma, Ideogram 등 64개 모델을 하나의 UI에서 선택 가능
- 이미지 편집: 특정 영역 지정 편집, 오브젝트 이동, 리라이팅, 카메라 변경, 배경 확장

- 생성 결과 그리드 레이아웃
- 로딩 중 상태 표시 방식 (실시간 프리뷰)

우리와 다른 점

**2. Magnific.ai**

한 줄 요약: AI 이미지 업스케일러 전문 — 저해상도 이미지를 최대 10K까지 확대하며 디테일을 새로 그려냄

**파트 1 — 텍스트 → 9컷 스토리보드 이미지 (일관성 있는 캐릭터)**

핵심 원리: "9-panel grid" 방식

GPT Image 2의 특성을 이용하는 2026년 현재 가장 검증된 방식:
```
[텍스트 기획 입력]
       ↓
[Claude API] → 9컷 분량 씬 설명 + 앵글 지시어 생성
       ↓
[GPT Image 2] → 9분할 그리드를 한 장의 이미지로 생성
       ↓
[결과물: 9컷 스토리보드 이미지 1장]
```
